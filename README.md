### 查詢成員、身分組、頻道、伺服器等 id

    /id member [member]
    /id role <role>
    /id channel <channel>
    /id guild

---

### 查詢機器人 ping

    /ping

---

### 新增/刪除身分組

    /role add <member> <role>
    /role remove <member> <role>

---

### 骰骰子

    /roll [min] [max]

### 匿名發言
